Class: HashPartitionResolver

HashPartitionResolver

new HashPartitionResolver(partitionKeyExtractor)

HashPartitionResolver implements partitioning based on the value of a hash function, allowing you to evenly distribute requests and data across a number of partitions.
Parameters:
Name Type Description
partitionKeyExtractor string | function If partitionKeyExtractor is a string, it should be the name of the property in the document to execute the hashing on. If partitionKeyExtractor is a function, it should be a function to extract the partition key from an object.
