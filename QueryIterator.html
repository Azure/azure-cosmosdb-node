QueryIterator

new QueryIterator(documentclient, query, options, fetchFunctions)

Represents a QueryIterator Object, an implmenetation of feed or query response that enables traversal and iterating over the response.
Parameters:
Name Type Description
documentclient object The documentclient object.
query SqlQuerySpec | string A SQL query.
options FeedOptions Represents the feed options.
fetchFunctions callback | Array.<callback> A function to retrieve each page of data. An array of functions may be used to query more than one partition.
