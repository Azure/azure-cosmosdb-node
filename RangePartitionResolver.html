RangePartitionResolver

new RangePartitionResolver(partitionKeyExtractor, partitionKeyMap, compareFunction)

RangePartitionResolver implements partitioning using a partition map of ranges of values to a collection link.
Parameters:
Name Type Description
partitionKeyExtractor string | function If partitionKeyExtractor is a string, it should be the name of the property in the document to execute the hashing on. If partitionKeyExtractor is a function, it should be a function to extract the partition key from an object.
partitionKeyMap Array The map from Range to collection link that is used for partitioning requests.
compareFunction function Optional function that accepts two arguments x and y and returns a negative value if x < y, zero if x = y, or a positive value if x > y.
